869. Reordered Power of 2

Here is a brief explanation of the question:

The problem is asking whether its possible to rearrange the digits of a given number in such a way that the resulting number (without any leading zeros) becomes a power of two. If this rearrangement is possible then function should return true; otherwise it should return false;

For example:

    If n is 46 you can rearrange the digits to form 64 which is a power of two Therefore the function should return true;

    If n is 65 it's not possible to rearrange the digits of n to form any number that is a power of two Therefore the function should return false;
